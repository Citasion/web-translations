搜索添加到该项目中的引用\n\n - 添加新的引用\n\n - 删除引用\n\n - 更新引用\n\n ### 手动添加新的引用\n\n 有多种方法可以向一个项目中添加引用，包括：\n\n - 从其他引用管理软件中导入数据\n\n - 解析PDF文件\n\n - 上传PDF论文\n\n - 复制粘贴BibTeX条目\n\n - 一键式引用导入\n\n - 手动添加引用\n\n 关于前五种添加引用的方法，请参考 <a href='/docs/docs/import-manage-papers'>这里</a>。\n\n 如果要手动添加一个引用，你需要：\n\n 1. 打开你需要的BibNext项目\n\n 2. 点击**向此项目添加引用**\n\n 3. 点击**手动添加**\n\n 4. 填写相关信息\n\n 对于那些添加到项目中的新的引用，你可以在任何与该项目关联的写作文件中使用（例如，谷歌文档、Word和Overleaf）。\n\n ### 更新引用\n\n 如果要更新引用，你需要：\n\n 1. 打开你需要的BibNext项目\n\n 2. 点击右上角的<img src='/static/images/support/dot-button.png' alt='dot' />按钮\n\n 3. 点击更新\n\n 4. 更新相关信息\n\n ### 删除引用\n\n 如果要删除引用，你需要：\n\n 1. 打开你需要的BibNext项目\n\n 2. 点击右上角的<img src='/static/images/support/dot-button.png' alt='dot' />按钮\n\n 3. 点击删除并确定\n\n <a name='manage-references-all'></a>\n\n ## 我如何在参考文献视窗中管理引用？\n\n <p> <a href='/references'> <img alt='Reference View' src='/static/images/docs/referenceView.png' /> </a> </p>\n\n 在参考文献视窗中，你可以：\n\n -
